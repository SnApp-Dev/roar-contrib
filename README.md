# Roar::Contrib
Collection of useful Roar contributions and extensions.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'roar-contrib'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roar-contrib
